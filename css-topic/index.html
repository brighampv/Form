CSS Topic Presentation!! - Spacing

To most people, they won't notice spacing between letters and between words other than the usual " " space like this one that we're used to.

So today I'll teach you about "letter-spacing" and "word-spacing"


So let's start with SPACES BETWEEN LETTERS

Just to be clear, to do this we will be using letter-spacing to make this happen
and this is measured in the following three ways below.

  • letter-spacing: XX px
    • formats the letter spacing using pixel values
  • letter-spacing: em
    • formats the letter spacing relative to the font
  • letter-spacing: normal
    • normal spacing between characters; this is the default

Now on to SPACES BETWEEN WORDS!!

This is important because some fonts have default spacing of "being too dang close to each other and it's really annoying" -Me from past experience

word-spacing can be very dynamic and can be changed using normal, px, or even cm!!

  • word-spacing: normal
    • normal spacing between characters; this is the default
  • word-spacing: XX px
    • formats the letter spacing using pixel values
  • word-spacing: XX cm
    • formats the letter spacing relative to the font
